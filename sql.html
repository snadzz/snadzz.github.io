This is SQL code for an insert statement

This code combines a left join between two tables, on a particular condition "cell c"
and uses the left function with as wildcat, as well as the right function likewise


Code:


insert into tblseooldcellc (date_rec, cell, o, Participant)
select a.date_rec, a.cell, a.o, b.Participant
from oldseodata a left join tblsaicomprefix b
on left(a.cell,3) like right(b.Prefix,3)
where b.participant = 'cellc';

This is SQL code for creating a table

The code below, creates a table from a left join between two tables with the
common data being wildcats and based on a specific condition "vodacom"


Code:


create table tblworlddataconvoda as
select b.EmailAddress, b.Landline, b.CellNumber, a.participant
from tblworlddatacon b left join tblsaicomprefix a
on left(b.CellNumber,2) like right(a.prefix,2)
where a.participant = 'vodacom';
